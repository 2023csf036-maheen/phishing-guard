**Kya Add Hoga:**

### Part A — Bigger Dataset
```
Free datasets available:
1. PhishNet    — 50,000+ phishing URLs
2. OpenPhish   — Real-time phishing feed
3. Alexa Top 1M— Legitimate URLs
4. APWG dataset— Verified phishing

In sab ko combine karo → 
100,000+ training samples
→ Accuracy 97% → 99%+
```

### Part B — BERT Model
```
BERT kya hai:
Google ka language model hai jo
URL ke har character ka context samjhta hai

Example:
"paypa1-secure-login-verify.tk/account"

BERT samjhega:
- "paypa1" ≈ "paypal" (similar meaning)
- "secure-login-verify" = urgency pattern
- ".tk" = suspicious ending
- Full URL ka combined meaning = PHISHING

Random Forest ye sab separately dekhta tha
BERT combined context dekhta hai → Better!
```

### Part C — Real-Time Learning
```
User feedback loop:
1. User URL scan karta hai
2. Result aata hai — PHISHING
3. User bolti hai "Ye actually safe tha"
4. Model ye feedback store karta hai
5. Automatically improve hota hai

Isko "Online Learning" kehte hain
```

## MODULE 4 — QR Code Scanner 📷
### (New Attack Vector — 2025 ka Trend)

**Kya hai:**
```
QR Code Phishing = "Quishing"
2024 mein 300% increase hua hai!

Attacker fake QR code banata hai
→ Scan karo → Phishing site pe le jata hai
→ Koi bhi URL scanner QR scan nahi karta!
```

**Tumhara Tool Kya Karega:**
```
1. Camera se QR code scan karo
2. QR se URL nikalo (decode)
3. PhishGuard URL scanner pe bhejo
4. Result — Safe/Dangerous
5. Bonus: QR image upload bhi kar sako

Use cases:
- Restaurant menu QR codes
- Shop payment QR codes  
- University notice board QR codes
- WhatsApp pe aaye QR codes
```

**Tech Stack:**
```
QR Decode:   OpenCV + pyzbar (Python)
Camera:      Flutter mobile camera API
Integration: Existing PhishGuard backend
```

**Kyun Unique:**
```
Pakistan mein JazzCash/EasyPaisa QR fraud
bohot common hai — koi tool detect nahi karta
Tumhara tool FIRST hoga Pakistan mein!
```
